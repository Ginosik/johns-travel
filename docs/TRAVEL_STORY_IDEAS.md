# Travel Story Ideas

This file is the planning backlog for future Conversante travel stories. The same ideas are mirrored in `src/data/storyIdeas.js` so the developer trip map can show them as planned draft nodes.

Each idea should eventually become a full story with English conversation text, Portuguese translations, vocabulary hints, cover media, and audio.

## Day 3 - Wrong Bus to the Right Beach

- Location: Campeche
- Activity: John takes the wrong bus and finds an unplanned beach stop.
- Language focus: Asking for directions, confirming stops, and explaining mistakes.
- Vocabulary theme: buses, stops, directions, beach access
- Conversation conflict: John is embarrassed, but the mistake turns into a useful discovery.
- Notes: Keep it light and practical: bus signs, where to get off, and asking someone nearby.
