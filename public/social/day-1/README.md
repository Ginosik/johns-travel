# John's Travel - Day 1 Instagram Carousel

Generated files for the Conversante social preview workflow.

- carousel.json: structured source for slides and visual rendering.
- caption.txt: Instagram caption draft.
- alt-text.txt: accessibility text for the carousel.
- preview.html and slides/: rendered by npm run social:render.
- day-1-instagram-package.zip: generated by npm run social:package.
